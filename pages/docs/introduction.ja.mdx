# Pulsate

**Pulsate** は、簡単に変更ができ、高速でユーザーフレンドリーな ActivityPub で実装された次世代分散型ソーシャルネットワーキングです。

<Callout type="warning">
  現在 Pulsate は v0 (開発途上バージョン) です。開発途上のため、ドキュメントは不完全な場合があります。安定版リリースに合わせて更新されます。
</Callout>
