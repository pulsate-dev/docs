# 開発に参加する

Pulsateはオープンソースプロジェクトです。誰でも開発に参加でき、参加する方法はいくつかあります。

## 参加の前に

Pulsateの開発に参加する前に、以下のことを確認してください。

1. **プロジェクトの概要**: 開発前にまだ読んでいなければPulsate の[プロジェクト概要](/docs/about) はぜひ一度読んでください。
2. **行動規範**: Pulsate は [行動規範](https://github.com/pulsate-dev/pulsate?tab=coc-ov-file) を制定しています。開発に参加する前に必ず読んでください。
3. **コミュニティ**: Pulsate は [Discord](https://link.pulsate.dev/discord) でコミュニケーションを取っています。開発に参加する場合はぜひ Discord に参加してください。

## 設計書

Pulsate はサードパーティでの開発を支援するために Pulsate の仕様を決めた設計書を全体公開しています。設計書 **Pulsate Specification** はいつでも [`spec.pulsate.dev`](https://spec.pulsate.dev/) から確認できます。

## 参加する

Pulsate には以下の方法で開発に参加できます。

### 機能の追加, 不具合の修正

Pulsate には [GitHub](https://link.pulsate.dev/github) で開発が行われています。貢献についてのガイドは [こちら](https://github.com/pulsate-dev/pulsate/blob/main/CONTRIBUTING.md) にあります。

### 議論への参加

新しい機能の追加や変更, 削除などの RFC (Request for Comments) は [GitHub Discussions](https://link.pulsate.dev/discussions) や [Discord](https://link.pulsate.dev/discord) で行われています。議論に参加する場合はぜひ参加してください。
